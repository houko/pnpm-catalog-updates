## 스타일 컨텍스트

### not-prose 클래스

산문 스타일에서 벗어나야 하는 컴포넌트에는 `<div className="not-prose">`를 사용하세요:

```mdx
<div className="not-prose">
  <Button href="/somewhere">클릭하세요</Button>
</div>
```

## 문서 작성 모범 사례

### 콘텐츠 구조

1. 메타데이터와 명확한 제목으로 시작
2. 서론에는 리드 단락 사용
3. 적절한 제목 계층으로 구성
4. 중요한 정보에는 Notes 추가
5. 실용적인 코드 예제 포함
6. 명확한 다음 단계로 마무리

### 작성 스타일

- 능동태 사용
- 간결하지만 완전하게 작성
- 모든 개념에 예제 포함
- 모든 코드 스니펫 테스트
- 일관된 용어 유지

### 구성

- 관련 주제를 함께 그룹화
- 상호 참조를 적극 활용
- 여러 진입점 제공
- 사용자의 여정 고려
- 검색 친화적인 제목 포함

## 완전한 문서 작성 워크플로

1. **계획**: 콘텐츠 구조 개요 작성
2. **작성**: 각 섹션에 적절한 컴포넌트 사용
3. **검토**: 완성도와 정확성 확인
4. **테스트**: 모든 예제가 작동하는지 확인
5. **반복**: 피드백을 바탕으로 개선

이제 세계 수준의 문서를 만드는 데 필요한 모든 도구를 갖추었습니다!
