# 코드 작성

문서에서 코드를 제시하는 기술을 익혀보세요. 구문 강조 사용법, 다중 언어 예제 생성, 효과적인 코드 블록 구성을 통해 개발자들이 솔루션을 이해하고 구현할 수 있도록 도움을 주는 방법을 알아보세요. {{ className: 'lead' }}

## 단일 코드 블록

자동 구문 강조가 있는 기본 코드 블록:

```javascript
// 구문 강조가 있는 JavaScript 예제
const greeting = '안녕, 세계!';
console.log(greeting);
```

위의 JavaScript 코드 블록은 다음 MDX 구문을 사용하여 생성됩니다:

````mdx
```javascript
const greeting = '안녕, 세계!';
console.log(greeting);
```
````

**Python 예제:**

```python
# Python 예제
def greet(name):
    return f"안녕, {name}!"

print(greet("세계"))
```

Python 코드 블록 MDX 구문:

````mdx
```python
def greet(name):
    return f"안녕, {name}!"

print(greet("세계"))
```
````

**Bash/Shell 명령어:**

```bash
# Bash/Shell 명령어
npm install @example/package
pnpm add dependency
```

Bash 코드 블록 MDX 구문:

````mdx
```bash
npm install @example/package
pnpm add dependency
```
````

## 다중 언어를 위한 CodeGroup

## API 엔드포인트 예제

API 문서의 경우 HTTP 메서드 태그를 사용하세요:

<CodeGroup tag="POST" label="/v1/users">

```bash {{ title: 'cURL' }}
curl -X POST https://api.example.com/v1/users \
  -H "Content-Type: application/json" \
  -d '{"name": "홍길동", "email": "hong@example.com"}'
```

```javascript {{ title: 'JavaScript' }}
const response = await fetch('https://api.example.com/v1/users', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    name: '홍길동',
    email: 'hong@example.com',
  }),
});
```

</CodeGroup>

위의 API 엔드포인트 예제는 다음 MDX 구문을 사용하여 생성됩니다. `tag`와 `label` 속성에 주목하세요:

````mdx

  -H "Content-Type: application/json" \
````

## 지원되는 언어

코드 블록은 다음을 포함한 다양한 프로그래밍 언어의 구문 강조를 지원합니다:

- **JavaScript/TypeScript**: `javascript`, `typescript`, `js`, `ts`
- **Python**: `python`, `py`
- **Shell scripts**: `bash`, `shell`, `sh`
- **기타 언어**: `json`, `yaml`, `xml`, `sql`, `css`, `html`, `markdown`, `diff`

**예제:**

```json
{
  "name": "example-package",
  "version": "1.0.0",
  "dependencies": {
    "react": "^18.0.0"
  }
}
```

JSON 코드 블록 MDX 구문:

````mdx
```json
{
  "name": "example-package",
  "version": "1.0.0",
  "dependencies": {
    "react": "^18.0.0"
  }
}
```
````
