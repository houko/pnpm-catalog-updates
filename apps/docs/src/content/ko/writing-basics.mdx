# 작성 기초

문서 작성의 기본 구성 요소를 익혀보세요. 이 가이드는 표준 Markdown 구문, 형식 옵션, 모든 문서에서 사용할 기본 요소들을 다룹니다. {{ className: 'lead' }}

## Markdown 기초

표준 Markdown 형식이 완전히 지원되며 모든 문서의 기초가 됩니다:

**굵은 텍스트**는 강조와 중요성을 위해  
*기울임 텍스트*는 미묘한 강조를 위해  
`인라인 코드`는 기술 용어, 명령어, 짧은 코드 스니펫을 위해

이들을 조합할 수 있습니다: **_굵고 기울임_** 또는 **굵은 텍스트와 `코드`**

## 텍스트 형식

### 강조와 굵은 텍스트

강조를 위해 별표나 밑줄을 사용하세요:

```markdown
_이것은 기울임 텍스트입니다_
_이것도 기울임입니다_

**이것은 굵은 텍스트입니다**
**이것도 굵습니다**

**_이것은 굵고 기울임입니다_**
```

### 코드와 기술 용어

인라인 코드, 변수, 또는 기술 용어에는 백틱을 사용하세요:

```markdown
패키지를 설치하려면 `npm install` 명령어를 사용하세요.
`NODE_ENV` 변수를 `production`으로 설정하세요.
`useState` 훅은 컴포넌트 상태를 관리합니다.
```

## 목록과 구성

### 순서 없는 목록

기능 목록, 요구사항, 또는 순서가 없는 항목에 완벽합니다:

- 주요 기능이나 요점
- 또 다른 중요한 항목
- 세 번째 고려사항
  - 중첩된 하위 요점
  - 추가 하위 세부사항
- 다시 주 레벨로

### 순서 있는 목록

단계별 지침, 절차, 또는 우선순위가 있는 항목에 사용하세요:

1. 프로세스의 첫 번째 단계
2. 중요한 세부사항이 있는 두 번째 단계
3. 세 번째 단계
   1. 구체적인 지침이 있는 하위 단계
   2. 또 다른 하위 단계
4. 마지막 단계

### 작업 목록

체크리스트와 진행 상황 추적에 좋습니다:

- [x] 완료된 작업
- [x] 완료된 또 다른 항목
- [ ] 대기 중인 작업
- [ ] 향후 개선사항

## 링크와 내비게이션

### 내부 링크

문서 내 다른 페이지로 링크하기:

```markdown
설정 지침은 [빠른 시작 가이드](/quickstart)를 참조하세요.
완전한 엔드포인트 세부정보는 [API 참조](/api)를 확인하세요.
```

예시:

- [명령어 참조 가이드](/command-reference)
- [문제 해결](/troubleshooting)
- [SDK 문서](/sdks)

### 외부 링크

외부 리소스로 링크하기:

```markdown
소스 코드는 [GitHub](https://github.com)을 방문하세요.
[공식 문서](https://example.com/docs)를 읽어보세요.
```

### 앵커 링크

페이지 내 특정 섹션으로 링크하기:

```markdown
위의 [텍스트 형식](#text-formatting) 섹션으로 이동하세요.
아래의 [모범 사례](#best-practices)를 참조하세요.
```

## 제목과 구조

적절한 제목 레벨로 명확한 문서 계층 구조를 만드세요:

```markdown
# 페이지 제목 (H1 - 메타데이터에서 자동으로 사용됨)

## 주요 섹션 (H2)

### 하위 섹션 (H3)

#### 세부 섹션 (H4)

##### 소규모 섹션 (H5)

###### 최소 섹션 (H6)
```

### 제목 모범 사례

- 페이지 제목에만 H1 사용 (메타데이터에서 처리됨)
- 섹션은 H2로, 하위 섹션은 H3로 시작
- 제목 레벨을 건너뛰지 말 것 (H2 → H4 금지)
- 제목을 설명적이고 스캔 가능하게 유지
- 문장 케이스 사용: "시작하기" (올바름)

## 표

구조화된 데이터를 위한 간단한 표:

| 기능     | 기본    | 프로     | 엔터프라이즈 |
| -------- | ------- | -------- | ------------ |
| 사용자   | 10명    | 100명    | 무제한       |
| 저장공간 | 1GB     | 10GB     | 100GB        |
| API 호출 | 1,000회 | 10,000회 | 무제한       |
| 지원     | 이메일  | 우선순위 | 24/7 전화    |

### 표 정렬

열 정렬 제어:

| 왼쪽 정렬    | 가운데 정렬  |  오른쪽 정렬 |
| :----------- | :----------: | -----------: |
| 텍스트       |    텍스트    |       텍스트 |
| 더 많은 내용 | 더 많은 내용 | 더 많은 내용 |

## 특수 문자

백슬래시를 사용하여 특수 Markdown 문자를 이스케이프하세요:

```markdown
\*기울임이 아님\*
\`코드가 아님\`
\# 제목이 아님
```
