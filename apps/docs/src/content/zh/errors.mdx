# 故障排除

常见问题和错误的解决方案。本页面帮助您快速解决使用 PCU 时遇到的问题。{{ className: 'lead' }}

## 常见错误

### 工作区未找到

如果 PCU 报告找不到工作区，请检查：

- `pnpm-workspace.yaml` 文件是否存在
- 文件路径配置是否正确
- 当前目录是否为工作区根目录

### 权限错误

权限相关的错误通常由以下原因引起：

- npm registry 访问权限
- 文件系统权限
- 网络防火墙设置

## 调试模式

使用 `--debug` 标志获取详细的调试信息。
