## 主要命令

### check

检查依赖项状态：

```bash
pcu check [options] [packages...]
```

### update

更新依赖项：

```bash
pcu update [options] [packages...]
```

### catalog

管理 catalog 配置：

```bash
pcu catalog <command> [options]
```

## 全局选项

- `--help, -h`: 显示帮助信息
- `--version, -v`: 显示版本号
- `--debug`: 启用调试模式
- `--config`: 指定配置文件路径
