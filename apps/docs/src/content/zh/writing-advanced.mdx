# 高级写作功能

掌握让您的文档专业而有效的高级功能。了解元数据、引导段落、样式上下文，以及区分优秀文档和良好文档的最佳实践。 {{ className: 'lead' }}

## 元数据和SEO

每个页面都应在顶部包含元数据：

```mdx
export const metadata = {
  title: '您的页面标题',
  description: '用于SEO和导航的简要描述。',
};

;
```

## 引导段落

使用`{{ className: 'lead' }}`使介绍段落脱颖而出：

```mdx
这是用更大、更突出的文本介绍主题的引导段落。 {{ className: 'lead' }}
```

## 样式上下文

### not-prose类

对需要脱离散文样式的组件使用`<div className="not-prose">`：

```mdx
<div className="not-prose">
  <Button href="/somewhere">点击我</Button>
</div>
```

## 文档最佳实践

### 内容结构

1. 从元数据和清晰标题开始
2. 使用引导段落进行介绍
3. 使用适当的标题层次组织
4. 为重要信息添加Notes
5. 包含实用的代码示例
6. 以清晰的后续步骤结束

### 写作风格

- 使用主动语态
- 简洁而完整
- 为每个概念包含示例
- 测试所有代码片段
- 保持术语一致性

### 组织方式

- 将相关主题组合在一起
- 大量使用交叉引用
- 提供多个入口点
- 考虑用户的使用旅程
- 包含对搜索友好的标题

## 完整的文档工作流程

1. **规划**：概述您的内容结构
2. **写作**：为每个部分使用适当的组件
3. **审查**：检查完整性和准确性
4. **测试**：验证所有示例都能工作
5. **迭代**：基于反馈进行改进
