# CI/CD 集成

将 PCU 集成到您的持续集成和部署流水线中。PCU 可以无缝集成到现有的 CI/CD 工作流中。{{ className: 'lead' }}

## GitHub Actions

示例 GitHub Actions 配置：

```yaml
name: Dependency Check
on: [push, pull_request]
jobs:
  check:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - run: npx pnpm-catalog-updates check
```
