# 工作区配置

配置 pnpm 工作区以充分利用 PCU 的功能。正确的工作区配置是使用 PCU 的前提条件。{{ className: 'lead' }}

## 基本配置

确保您的 `pnpm-workspace.yaml` 文件正确配置：

```yaml
packages:
  - 'apps/*'
  - 'packages/*'
  - 'libs/*'

catalogs:
  default:
    react: ^18.0.0
    typescript: ^5.0.0
  dev:
    eslint: ^8.0.0
    prettier: ^3.0.0
```

## Catalog 最佳实践

了解如何有效组织您的 catalog 配置以获得最佳的依赖管理体验。
